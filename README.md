# CoolBeatProjects

自动抓取通过 “备份到云端” 功能泄露到公网的玩酷电音工程，整理为 JSON 并利用 [Prettier](https://prettier.io/) 格式化

## Usage

```
./grab.sh 1 300000 # grab from ID 1 to 300000
```
